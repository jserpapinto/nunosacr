# Instruções

## Requisitos
* php5.6+
* composer
* php artisan
* algumas bibliotecas php: GD, mbstring..

## Configuração
* No diretório do projeto, executar: 
	* composer install
	* php artisan serve
* No browser
	* http://localhost:8000 -> index
	* http://localhost:8000/admin -> backoffice

		* pass: password
